# Megapower Equipment & Engineering

Static website for Megapower Equipment & Engineering.

## Local Preview

```sh
npm run dev
```

Open http://127.0.0.1:8000/.

## Vercel Deployment

Use these Vercel project settings:

- Framework Preset: Other
- Build Command: None
- Output Directory: `.`
- Install Command: None

The project root contains the deployable `index.html`, `styles.css`, `script.js`, and `assets/` files.
